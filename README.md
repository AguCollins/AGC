# Agu Collins Chidera · Frontend Developer Portfolio

React-based portfolio website built with Vite, Bootstrap, and modern frontend practices.

## Features

- Fully responsive design
- Dark / Light mode toggle
- Animated skill bars and counters
- Particle background in hero
- Carousel projects and testimonials
- Contact form with validation (react-hook-form)
- Toast notifications
- SEO-friendly with react-helmet-async
- High performance and accessibility

## Tech Stack

- React 18
- Vite
- Bootstrap 5 + react-bootstrap
- AOS (scroll animations)
- Typed.js (hero typing effect)
- react-toastify
- react-scroll (smooth navigation)
- react-hook-form

## Installation

```bash
npm install
npm run dev
